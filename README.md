# NiceGui_DR
## main.py
    Este programa crea una calculadora básica con interfaz gráfica usando NiceGUI, que permite realizar las operaciones de:

            Suma

            Resta

            Multiplicación

            División

        El sistema controla errores de ingreso.
        los datos que son ingresados como num1 y num2 ejecutarán la operación que se le asigne, debido a que cada una ya cuenta con el proceso que debe llevarse a cabo para ,mostrar el resultado directamente en pantalla después de el proceso matemático.
